Website skeleton
================

This repository contains a minimal website skeleton.

Quick start
-----------

- Open [index.html](index.html) in your browser.
- Edit `css/styles.css` and `js/script.js` to customize styles and behavior.

Next steps
----------

- Add assets in an `assets/` or `images/` folder.
- Integrate a build tool (Vite, webpack) if you need bundling.
- Deploy to GitHub Pages or any static host.
