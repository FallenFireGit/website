Portfolio Website Example
================

This repository contains my first website without the use of external help from generative AI (besides the inline editor that I later learned does count as AI, but after learning it counts as AI, I tried to limit its use). As a person who mostly does hardware website UI was a relatively new thing for me, I have before messed around with inspect element to change some numbers for devious purposes like my grades (woops) or other things like the Google landing page to make it say elgoog. So this project wasn't unfamiliar, but definitely new territory. Also, we frequently look at website builders like Wix or Squarespace, but few people actually know how to build a website themselves from raw code. That was the goal of this project: to learn how a website is actually made (and learned it's actually surprisingly easy for at least the front-end) instead of relying on website builders or chats to make them for me. Plus, I think it is also very obvious when a website is vibe-coded anyway and takes away a little bit of that human creativity that the world needs more of. While vibe-coding, web builders, and no-code designs may be the future, at least I can say (to an extent) that I have made my own without these tools for the sake of just learning and creating. - FallenFire
